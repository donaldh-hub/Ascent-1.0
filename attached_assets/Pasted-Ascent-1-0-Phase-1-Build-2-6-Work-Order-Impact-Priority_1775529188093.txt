==================================================

1. WORK ORDER IMPACT ENGINE (CORE LAYER)
   ==================================================

Create a centralized Work Order Impact Engine service.

This engine must:

accept work order data
calculate impact scores per work order
aggregate impact at category level
aggregate impact at property level
aggregate impact at regional level

OUTPUT:

Work Order Impact Score (per record)
Category Impact Score
Property Impact Score
Regional Impact Ranking

==================================================
2. IMPACT INPUTS (REQUIRED)
===========================

Each record must contribute to impact using:

A. TIME WEIGHT

* days in stage (or days open)
* aging severity

B. CATEGORY WEIGHT

Define base weights:

Maintenance = 1.0
HVAC = 1.2
Plumbing = 1.1
Electrical = 1.1
Appliance = 1.2
Turn/Make-Ready = 1.3
General = 0.8

(Allow future configuration)

C. BLOCK STATUS

Blocked = multiplier (1.5x – 2.0x)
Stalled = multiplier (1.3x)
Normal = baseline

D. PRIORITY SIGNAL

Critical = 2.0x
High = 1.5x
Medium = 1.0x
Low = 0.7x

E. REPEAT ISSUE SIGNAL (if detectable)

If same unit/category repeats:

* additional multiplier

==================================================
3. WORK ORDER IMPACT SCORE (FORMULA)
====================================

Impact Score =
(Time Open or Days in Stage)
× (Category Weight)
× (Block Multiplier)
× (Priority Multiplier)
× (Repeat Multiplier if applicable)

Must output:

numeric score
severity tier (low / medium / high / critical)
explainability string

Example:

"High impact due to 10 days open in HVAC with critical priority"

==================================================
4. CATEGORY IMPACT AGGREGATION
==============================

Aggregate impact across categories:

For each category:

sum all Work Order Impact Scores
calculate average impact
calculate blocked ratio

Output:

Category Impact Score
Top contributing work orders
Explanation:

"Maintenance is highest impact due to backlog and aging work orders"

==================================================
5. PROPERTY IMPACT ENGINE
=========================

Aggregate impact by property:

Sum all Work Order Impact Scores per property
Rank properties by total impact

Output:

Property Impact Score
Property Rank
Primary issue category

Example:

"Cedars of Chili is highest impact due to maintenance backlog"

==================================================

Rank top issues across portfolio

Output TOP 3:

#1 Priority Issue

#3 Priority Issue

Reason

SECTION 1 — CLICKABLE METRIC CARDS (TOP STRIP)
==============================================

The following cards must be interactive:

* Flow
* Risk
* Execution
* Improvement

CURRENT ISSUE:
Cards highlight but do nothing.

REQUIRED FIX:
Each card must REVEAL its drivers.

==================================================
INTERACTION BEHAVIOR (MANDATORY)
================================

Clicking a metric card must:

EITHER:

* expand a detail panel directly below the card

OR:

* scroll to a dedicated section on the same page

DO NOT:

* navigate to a new page
* reload the screen

==================================================
METRIC REVEAL CONTENT
=====================

FLOW MUST SHOW:

* top bottleneck stage
* number of stuck items
* aging time
* affected workflows

RISK MUST SHOW:

* critical items (count + age)
* overdue items
* red workflows
* missing documentation tied to critical items

EXECUTION MUST SHOW:

* movement consistency
* stale items
* completion rate
* responsiveness issues

IMPROVEMENT MUST SHOW:

* trend direction (up/down)
* recovery vs decline
* completion momentum

Each reveal section must include:

* short headline (WHY)
* supporting bullet points

SECTION 2 — PROPERTY CARD INTERACTION UPGRADE
=============================================

Each property card must behave as a MINI CONTROL TOWER.

==================================================
DEFAULT STATE (COLLAPSED)
=========================

Show:

* property name
* health score + stoplight
* flow / risk / improvement / execution
* signals (critical items, bottleneck, docs)
* insight text
* supervisor action block (visible)

==================================================
EXPANDED STATE (ON CLICK)
=========================

Clicking the card must:

* expand inline (NO navigation)
* transform into detailed control view

Expanded content must include:

* TOP ISSUE (primary problem)
* WORKFLOW BREAKDOWN (health drivers)
* AFFECTED UNITS (list)
* DOCUMENT WARNINGS (evidence gaps)
* WORKFLOW-LEVEL HEALTH

SupervisorActionBlock remains visible.
